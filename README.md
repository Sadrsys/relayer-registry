# Tornado Cash relayer registry contract [![Build Status](https://github.com/tornadocash/relayer-registry/workflows/build/badge.svg)](https://github.com/tornadocash/relayer-registry/actions)

## Dependencies

1. node 12
2. yarn

## Start

```bash
$ yarn
$ yarn test
```

## Deploying

Deploy to Kovan:

```bash
$ yarn deploy:kovan
```
